[](https://docs.polygonscan.com/api-endpoints/<#get-pol-balance-for-a-single-address>)
Get POL Balance for a Single Address
Returns the POL balance of a given address.
Copy```
https://api.polygonscan.com/api
  ?module=account
  &action=balance
  &address=0x5A534988535cf27a70e74dFfe299D06486f185B7
  &apikey=YourApiKeyToken
```

> Try this endpoint in your [**browser** ](https://docs.polygonscan.com/api-endpoints/<https:/api.polygonscan.com/api?module=account&action=balance&address=0x5A534988535cf27a70e74dFfe299D06486f185B7&apikey=YourApiKeyToken>)üîó
RequestResponse
Query Parameters
Parameter
Description
address
the `string` representing the address to check for balance
tag
the `string` pre-defined block parameter, either `earliest`, `pending` or `latest`
Sample response
Copy```
{
  "status":"1",
  "message":"OK",
  "result":"9996128487000000000"
}
```

üìñ **Tip:** The `result` is returned in [**wei.**](https://docs.polygonscan.com/api-endpoints/<https:/polygonscan.com/unitconverter>)
Convert POL units using our [**Unit Converter.**](https://docs.polygonscan.com/api-endpoints/<https:/polygonscan.com/unitconverter>)

Copy```
https://api.polygonscan.com/api
  ?module=account
  &action=balancemulti
  &address=0x5A534988535cf27a70e74dFfe299D06486f185B7,0x54bA15efe1b6D886bA4Cd5C5837240675BD0D43a,0x39842a0Fe638cc956b76A49E918c30d818708BA0
  &tag=latest
  &apikey=YourApiKeyToken
```

> Try this endpoint in your [**browser**](https://docs.polygonscan.com/api-endpoints/<https:/api.polygonscan.com/api?module=account&action=balancemulti&address=0x5A534988535cf27a70e74dFfe299D06486f185B7,0x54bA15efe1b6D886bA4Cd5C5837240675BD0D43a,0x39842a0Fe638cc956b76A49E918c30d818708BA0&tag=latest&apikey=YourApiKeyToken>) üîó
RequestResponse
Query Parameters
Parameter
Description
address
the `strings` representing the addresses to check for balance, separated by `,` commas
tag
the `string` pre-defined block parameter, either `earliest`, `pending` or `latest`
Sample Response
Copy```
{
  "status":"1",
  "message":"OK",
  "result":[

      "account":"0x54bA15efe1b6D886bA4Cd5C5837240675BD0D43a",

      "account":"0x39842a0Fe638cc956b76A49E918c30d818708BA0",
      "balance":"100761697447268339"

  ]
}
```
